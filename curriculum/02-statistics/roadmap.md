# 02-statistics — 완주형 로드맵 (데이터 해석의 기초 체력)

> 목표: 공식을 외우는 통계가 아니라, **데이터를 신뢰성 있게 읽고 의사결정에 쓰는 능력**을 만든다.

---

## 1) 챕터 완료 기준 (Definition of Done)

아래를 모두 충족하면 `02-statistics` 완료:

- [ ] 기술통계(평균/중앙값/분산/표준편차) 의미를 설명 가능
- [ ] 분포(정규/왜도/이상치) 해석 가능
- [ ] 표본/모집단, 신뢰구간, 가설검정 흐름 이해
- [ ] 상관관계와 인과관계 차이를 사례로 설명 가능
- [ ] Python으로 통계 계산/시각화 실습 8개 이상 완료
- [ ] 미니 프로젝트 2개 완료
- [ ] 평가 루브릭 통과 + 회고 제출

---

## 2) 학습 목표 (Learning Outcomes)

완료 시 학습자는 다음을 할 수 있어야 한다.

1. 데이터 요약 지표를 계산하고 비즈니스 의미로 해석한다.
2. 분포 형태/이상치를 찾아 분석 리스크를 설명한다.
3. 검정 전제(독립성/표본크기/분포)를 확인한다.
4. p-value를 오해 없이 해석한다.
5. A/B 테스트 결과를 구조화해서 보고한다.
6. 통계 결과를 과장하지 않고 한계와 불확실성을 함께 제시한다.

## 6) 자주 나는 실수 & 교정법

1. **평균 하나로 전체를 판단**
   - 교정: 중앙값/IQR/분포 그래프 반드시 함께 확인
2. **p-value를 효과크기로 오해**
   - 교정: 효과크기(차이 크기)와 분리해서 보고
3. **상관관계를 인과로 단정**
   - 교정: 교란변수/실험설계 필요성을 명시
4. **전제조건 확인 없이 검정 수행**
   - 교정: 검정 전 체크리스트(표본크기/독립성/분포) 선행
