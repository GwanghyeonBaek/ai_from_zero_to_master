# 08-mlops — 운영 배포 로드맵

> 목표: 모델을 노트북 실험에서 끝내지 않고, 배포/모니터링/재학습까지 운영 가능한 형태로 완성한다.

## 참고자료
- Primary: Made With ML (MLOps), Google MLOps Whitepaper
- Secondary: Full Stack Deep Learning

## 학습목표
1. 추론 API를 컨테이너 기반으로 배포한다.
2. 모델/데이터 버전 관리와 CI/CD 파이프라인을 구성한다.
3. 서비스 지표 + 모델 품질 지표를 모니터링한다.
4. 드리프트 탐지와 재학습 트리거를 설계한다.
5. 운영 runbook과 incident 대응 절차를 문서화한다.

## 필수 실습
- [ ] 서빙 API 컨테이너화
- [ ] CI 파이프라인 구축
- [ ] 모델 레지스트리/버전 태깅
- [ ] 모니터링 대시보드 구성
- [ ] 드리프트 알림 규칙 설정
- [ ] 재학습 시뮬레이션

## 평가(100)
- 배포 안정성: 25
- 자동화(CI/CD): 20
- 모니터링/알림: 20
- 드리프트/재학습 설계: 20
- 운영 문서/재현성: 15

Pass: 75
