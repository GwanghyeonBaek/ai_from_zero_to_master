# Learning Contract (학습 계약서)

## 1) 목표
- 나는 AI/Data 역량을 초보부터 실무/마스터 수준까지 완주한다.
- 목표 기간: ____년 __월 __일 ~ ____년 __월 __일

## 2) 실행 약속
- 주당 학습 횟수: 최소 5회
- 1회 학습 시간: 60~90분
- 학습 증빙: 매회 로그 + 주 3회 이상 커밋

## 3) 중단 방지 규칙
- 연속 2일 공백 금지
- 슬럼프 발생 시 최소 20분 마이크로 학습으로 복귀
- 막힘 발생 시 25분 내 질문/문서 검색/이슈 등록

## 4) 품질 기준
- 이론 요약 10줄 + 코드 실습 + 회고 3줄
- 챕터 종료 시 미니 프로젝트 1개
- 월말 포트폴리오 업데이트 1회

## 5) 보안/윤리 규칙
- 비밀정보/토큰/개인정보를 코드에 커밋하지 않는다.
- 생성형 AI 사용 시 출처와 검증 여부를 표시한다.
- 저작권/라이선스 조건을 확인한다.
