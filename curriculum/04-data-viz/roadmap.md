# 04-data-viz — 완주형 로드맵 (데이터를 설명 가능한 인사이트로)

> 목표: 차트를 그리는 수준을 넘어, **의사결정을 돕는 시각적 스토리텔링**을 수행한다.

## 1) 완료 기준 (DoD)
- [ ] 차트 선택 원칙(비교/추세/구성/분포) 설명 가능
- [ ] Matplotlib/Seaborn으로 핵심 차트 제작 가능
- [ ] 잘못된 시각화 사례를 교정 가능
- [ ] 실습 10개 이상 완료
- [ ] 미니 프로젝트 2개 완료
- [ ] 리포트 + 회고 제출

## 2) 학습목표
1. 질문에 맞는 차트를 선택한다.
2. 축/스케일/색상/주석으로 오해를 줄인다.
3. 한 장표로 핵심 메시지를 전달한다.
4. 재현 가능한 시각화 코드와 리포트를 작성한다.

## 3) 주차별 진행 (2~3주)
### Week 1
- 기본 차트(막대/선/히스토그램/박스플롯)
- 실습: 카테고리 비교, 시계열 추세

### Week 2
- 고급 시각화(히트맵/산점도/페어플롯)
- 시각적 인코딩/색채/레이아웃
- 실습: 상관관계 탐색, 이상치 시각화

### Week 3
- 스토리텔링 슬라이드/리포트
- 미니 프로젝트 2개

## 4) 필수 과제 (10개)
- [ ] 막대 차트 2종
- [ ] 시계열 선 그래프
- [ ] 히스토그램 + KDE

- [ ] 최종 미니 프로젝트
