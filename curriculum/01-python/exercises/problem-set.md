# 01-python 문제집 (필수 12문제)

## Basic (1~4)
1. 섭씨를 화씨로 변환하는 함수 작성
2. 점수 리스트의 평균/최댓값/최솟값 출력
3. 문자열에서 모음 개수 세기
4. 1~100 중 3의 배수 합 구하기

## Intermediate (5~8)
5. 학생 점수 딕셔너리에서 상위 3명 추출
6. 중복 이메일 제거 + 도메인별 개수 집계
7. 잘못된 입력을 처리하는 계산기 구현
8. CSV 성적 파일을 읽어 과목별 평균 계산

## Applied (9~12)
9. 학습 로그 파일에서 총 공부시간 계산
10. 일자별 누적 학습시간 리포트 생성
11. 로그 파일에서 가장 많이 나온 에러 메시지 추출
12. 메뉴형 CLI(조회/저장/종료) 프로그램 작성

## 제출 형식
- 각 문제별 `q01.py ... q12.py`
- 실행 예시를 `answers.md`에 기록
