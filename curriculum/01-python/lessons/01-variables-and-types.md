# Lesson 01 — 변수와 타입

## 학습목표
- 변수/타입의 역할을 설명한다.
- `int`, `float`, `str`, `bool`을 구분해 사용한다.

## 핵심 개념
- 변수는 데이터를 담는 이름표
- 타입은 데이터의 성질
- 타입이 다르면 가능한 연산도 달라진다.

## 예제
```python
name = "Alex"
age = 20
height = 173.5
is_student = True
print(type(name), type(age), type(height), type(is_student))
```

## 자주 하는 실수
- 숫자를 문자열로 받아놓고 연산하려다 오류 발생

## 체크 문제
1. `"10" + "20"` 결과는?
2. `int("10") + int("20")` 결과는?
