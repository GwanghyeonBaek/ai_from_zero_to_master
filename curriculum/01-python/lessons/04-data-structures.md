# Lesson 04 — 자료구조(리스트/딕셔너리/셋/튜플)

## 학습목표
- 상황에 맞는 자료구조를 선택한다.

## 선택 기준
- 리스트: 순서 중요, 중복 허용
- 딕셔너리: key-value 조회
- 셋: 중복 제거
- 튜플: 변경 없는 고정 데이터

## 예제
```python
scores = [80, 90, 90, 70]
unique_scores = set(scores)
student = {"name": "Alex", "score": 90}
coord = (37.5, 127.0)
```
