# Lesson 03 — 함수

## 학습목표
- 함수를 정의하고 호출한다.
- 파라미터와 반환값을 활용한다.

## 핵심 개념
- 함수는 재사용 가능한 코드 블록
- 입력(파라미터) -> 처리 -> 출력(반환값)

## 예제
```python
def add(a, b):
    return a + b

result = add(3, 7)
print(result)
```

## 자주 하는 실수
- `return` 없이 결과를 기대함
- 함수 내부 변수와 외부 변수 범위를 혼동

## 미션
- 점수를 받아 A/B/C를 반환하는 함수 작성
