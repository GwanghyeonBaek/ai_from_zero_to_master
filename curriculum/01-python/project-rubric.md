# 01-python Mini Project Rubric

총점 100점 / 통과 75점

1) 문제 정의 명확성 (20)
- 해결하려는 문제와 입력/출력이 명확한가?

2) 코드 구조 (20)
- 함수 분리, 파일 구조, 재사용성이 적절한가?

3) 안정성 (20)
- 예외처리/입력검증이 적용되어 있는가?

4) 재현성 (20)
- README 기준으로 다른 사람이 실행 가능한가?

5) 회고 품질 (20)
- 어려움/개선점/다음 계획이 구체적인가?

## 감점 기준
- 실행 불가: -30
- README 부재: -20
- 예외처리 없음: -10
- 하드코딩 과다: -10
