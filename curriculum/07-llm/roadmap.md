# 07-llm — 실전 LLM 응용 로드맵

> 목표: 프롬프트/검색증강(RAG)/평가를 결합해 재현 가능한 LLM 애플리케이션을 구축한다.

## 참고자료(기록)
- Primary: Google MLCC LLM, OpenAI Cookbook
- Secondary: Hugging Face Course, Stanford CS324

## 학습목표
1. 프롬프트 설계를 구조화하고 재사용 템플릿으로 관리한다.
2. RAG 파이프라인(query→retrieve→generate)을 구현한다.
3. LLM 출력 품질을 정량/정성 지표로 평가한다.
4. 환각/안전/비용 리스크를 점검하고 완화한다.
5. 증빙 가능한 LLM 실험 패키지를 제출한다.

## 필수 실습
- [ ] 프롬프트 변형 실험 3종
- [ ] RAG 인덱싱/검색/생성 파이프라인 구축
- [ ] 평가셋 기반 응답 품질 비교
- [ ] 환각 사례 수집 및 완화 실험
- [ ] 프로젝트 A/B 완료

## 평가(100)
- 프롬프트 설계 품질: 20
- RAG 구현 완성도: 25
- 평가/벤치마크 신뢰성: 20
- 안전/환각 대응: 20
- 재현성/증빙: 15

Pass: 75
