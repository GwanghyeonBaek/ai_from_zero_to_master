# Project Track (Beginner → Pro)

## Level 1 (Beginner)
- Titanic 생존 예측
- 서울 공공데이터 EDA
- 주식/환율 간단 시계열 시각화

## Level 2 (Intermediate)
- 고객 이탈 예측
- 추천시스템 기초
- 텍스트 감성분석

## Level 3 (Advanced)
- RAG 기반 Q&A 시스템
- LLM 파인튜닝(PEFT/LoRA)
- 모델 배포 + 모니터링(MLOps)

각 프로젝트는 아래 4가지를 반드시 포함:
1. 문제정의
2. 데이터 설명/전처리
3. 모델링/평가
4. 결과 해석 및 개선안
